# DUST Project Page

Static academic project page for:

**An On-Orbit Star Tracker Benchmark for Resident Space Object Detection and Attitude Estimation**

This site is adapted from the SpaceSense-Bench academic project page layout.

## What is included

- `index.html` - main project page
- `static/images/` - cropped figures extracted from the paper PDF
- `static/pdf/dust-scientific-data-2026.pdf` - local copy of the paper PDF
- `static/css/`, `static/js/`, `static/icon/` - template assets
- `.nojekyll` - keeps GitHub Pages from ignoring folders/files

## Before publishing

Check these links in `index.html`:

- Paper DOI: `https://doi.org/10.1038/s41597-026-07736-9`
- Dataset DOI: `https://doi.org/10.5281/zenodo.20255672`
- Code URL: currently set to `https://github.com/vithurshansuthakar/DUST`; change it if the repository name differs.

## Deployment option A: project page repo

1. Create a new GitHub repository, for example `DUST`.
2. Copy all files from this folder into the repository root.
3. Commit and push.
4. In GitHub: **Settings → Pages → Build and deployment → Deploy from a branch → main / root**.
5. Your URL will be similar to: `https://vithurshansuthakar.github.io/DUST/`.

## Deployment option B: add under your existing website

1. Copy this folder into your `vithurshansuthakar.github.io` repository as `dust/`.
2. Commit and push.
3. Your URL will be: `https://vithurshansuthakar.github.io/dust/`.
